自作エンジンでゲームを制作しています。
制作途中段階です。
※　現在2Dが表示されなくなっているため、
　　実行すると画面が黒く塗りつぶされます(タイトルです)。
※  継承を使ったことを後悔してます。

KyotEngine/Game.sinを開き、 「F5」を押すことで実行できます。

使用している素材はすべて自作です。

制作使用ツール

VisualStudio2017
3Dモデル::		3dsMax
2D,テクスチャ::		FireAlpaca64
　
操作方法

挙動確認コントローラー	[Logicool GamePad F310]

コントローラーB		タイトルからゲームへの遷移
コントローラーA		持つ/置く
コントローラーX		洗う/切る
左スティック		移動
